# CricBee

CricBee is a cricket platform with live scoring, streaming, tournament management, and fan engagement.  
This repository contains a **FastAPI**-based backend and a **React + Vite** frontend.

---

## Tech Stack

- **Backend**: Python, FastAPI, SQLAlchemy, Celery
- **Frontend**: React, Vite, Tailwind CSS, React Router, Redux Toolkit
- **Database / Services**: PostgreSQL, Redis (via `docker-compose.yml`)

---
# Features

#Core Cricket Management
Tournament Management - Create, manage, and organize cricket tournaments
Team & Player Management - Comprehensive player profiles a
Live Scoring - Real-time match scoring 
Fixture Management - Automated scheduling and fixture generation
Point Tables - Dynamic standings and tournament rankings

#Multi-Role System

Organizers - Tournament creation, management, and financial oversight
Club Managers - Team management, player registration, and match participation
Players - Profile management, statistics tracking, and match participation
Fans - Live match viewing, score updates, and engagement features
Admin - System administration, financial management, and user oversight


#Real-time Features
Live Scoring - WebSocket-based real-time score updates
Chat System - In-app messaging and communication
Notifications - Real-time alerts and updates
